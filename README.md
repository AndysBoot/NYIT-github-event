This repository is designed to help you learn the basics of Git and GitHub. We'll guide you through downloading the project, making changes, and creating your own version.

What is Git and GitHub? Git is a version control system that tracks changes in files, and GitHub is a website that hosts those repositories online.

Instructions:
To copy this repo, first make sure you have Git installed on your computer. Then go to the main page of this repository and click the button labeled "Code," copy the URL, and in your command line type git clone followed by the URL.
